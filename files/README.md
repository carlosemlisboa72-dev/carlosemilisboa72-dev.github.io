## 🔧 Wie Sie die Dateien auf GitHub hochladen

### **Option 1: Über GitHub Web Interface (einfach)**

1. Gehen Sie zu: https://github.com/carlosemlisboa72-dev/carlosemilisboa72-dev.github.io
2. Klicken Sie auf "Add file" → "Upload files"
3. Drag & Drop alle `.html` Dateien hierher
4. Schreiben Sie eine Commit-Nachricht: "Update: New website structure with 4 themes, legal pages"
